# Oddments

Windows에서 **특정 창·모니터 화면을 캡처**하고, **키워드 OCR**과 **템플릿 이미지 매칭**으로 이벤트를 감지하면 **알림음**을 재생하는 데스크톱 앱입니다.  
미리보기·설정은 **Flet**(Material Design 3) GUI로 제공합니다.

- **OCR**: **RapidOCR** 한 종류를 사용합니다. 최초 실행 시 한글 인식용 ONNX·사전 등 필요한 파일을 자동으로 내려받을 수 있습니다.
- **설정 저장**: `oddments_settings.json` — 소스 실행 시 프로젝트 루트, **exe 실행 시 exe와 같은 폴더**에 저장됩니다.

## 아두이노 연동 (Windows)

시리얼(COM)로 연결된 **아두이노**에, 지정한 키가 눌리거나 떼질 때마다 한 줄씩 ASCII로 전송할 수 있습니다. (예: `D,<가상키코드>` / `U,<가상키코드>`, 줄 끝 `\n`)  
설정에서 COM 포트·보드레이트·감시할 키 목록을 지정하고, 필요하면 **창 포커스를 얻거나 잃을 때** 별도 키 이벤트를 보내는 옵션도 켤 수 있습니다. 감지·알림과 별도로 외부 장치와 연동할 때 쓰입니다.

## 웹 화면 송출

같이 캡처하는 화면을 **로컬 HTTP(S) 서버 + WebRTC**로 브라우저에 송출할 수 있습니다.  
기기나 다른 PC의 브라우저에서 접속해 시청하고, Windows에서는 **시스템 재생 오디오(WASAPI loopback)** 를 함께 보낼 수 있습니다. 포트·화질·오디오 출력 장치·HTTPS(인증서·개인키 PEM) 등은 **웹 화면 송출 설정**에서 지정합니다. 외부에서 보려면 공유기 포트포워딩·방화벽 등 네트워크 설정이 필요할 수 있습니다.
